### Learning Summary

#### Feature Engineering
Feature engineering is the process of creating, selecting, transforming, and improving features (variables) from raw data to enhance the performance of machine learning models. It is one of the most important steps in the machine learning pipeline, as the quality of features directly impacts model accuracy.

#### Feature Selection
Feature selection involves choosing the most relevant features from a dataset while removing redundant or irrelevant ones.

**Benefits:**
- Reduces model complexity.
- Improves model accuracy.
- Decreases training time.
- Helps prevent overfitting.

#### Feature Extraction
Feature extraction creates new features by combining or transforming existing ones while preserving essential information.

**Examples:**
- Principal Component Analysis (PCA)
- Text vectorization (TF-IDF)
- Image feature extraction

#### Feature Transformation
Feature transformation modifies existing features into a more suitable format for analysis or model training.

**Common Techniques:**
- Log Transformation
- Square Root Transformation
- Normalization
- Standardization

#### Feature Scaling
Feature scaling ensures that numerical features are on a similar scale, improving the performance of many machine learning algorithms.

**Common Methods:**
- Min-Max Scaling
- Standard Scaling (Z-score Normalization)
- Robust Scaling

#### Encoding Categorical Variables
Machine learning algorithms require numerical input, so categorical data must be converted into numerical form.

**Common Encoding Techniques:**
- Label Encoding
- One-Hot Encoding
- Ordinal Encoding
