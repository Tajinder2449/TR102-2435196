### Learning Summary

#### Introduction to Machine Learning (ML)

Machine Learning (ML) is a branch of Artificial Intelligence that enables computers to learn from data and improve their performance without being explicitly programmed. ML algorithms identify patterns from existing data and use them to make predictions or decisions.

#### Types of Machine Learning

Machine Learning is mainly classified into three types:

**1. Supervised Learning**
- Learning method where the model is trained using labeled data.
- The algorithm learns the relationship between input features and output labels.

**2. Unsupervised Learning**
- Learning method where the model works with unlabeled data.
- It identifies hidden patterns, groups, or structures within the data.

**3. Reinforcement Learning**
- Learning method where an agent learns by interacting with an environment.
- It improves performance through rewards and penalties.

---

## Supervised Learning

#### Introduction to Supervised Learning

Supervised learning is a machine learning approach in which a model is trained using a labeled dataset. The algorithm learns from input-output pairs and uses the learned patterns to predict outcomes for new, unseen data.

#### Working of Supervised Learning

The process of supervised learning involves:

1. **Data Collection**
   - Gathering labeled data containing input features and expected outputs.

2. **Data Preprocessing**
   - Cleaning and transforming data before training.

3. **Model Training**
   - The algorithm learns patterns and relationships from training data.

4. **Model Evaluation**
   - Performance is measured using suitable evaluation metrics.

5. **Prediction**
   - The trained model predicts outputs for new data.

---

### Types of Supervised Learning

#### 1. Classification

Classification is used when the output belongs to a specific category or class.

**Examples:**
- Spam email detection.
- Disease prediction.
- Image recognition.

**Common Algorithms:**
- Logistic Regression
- Decision Tree
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Random Forest

---

#### 2. Regression

Regression is used when the output is a continuous numerical value.

**Examples:**
- House price prediction.
- Sales forecasting.
- Weather prediction.

**Common Algorithms:**
- Linear Regression
- Polynomial Regression
- Decision Tree Regression
- Random Forest Regression
