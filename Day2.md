### Learning Summary

#### Basics of Python
- Python is a high-level, interpreted, and easy-to-learn programming language.
- It is widely used in data analysis, machine learning, artificial intelligence, web development, and automation.
- Python's simple syntax makes it suitable for beginners as well as professionals.

#### Python Data Types
Python provides various built-in data types to store different kinds of values.

- **Integer (`int`)** – Whole numbers (e.g., `10`, `-5`)
- **Float (`float`)** – Decimal numbers (e.g., `3.14`)
- **String (`str`)** – Sequence of characters (e.g., `"Hello"`)
- **Boolean (`bool`)** – Logical values (`True` or `False`)
- **Complex (`complex`)** – Complex numbers (e.g., `2+3j`)

#### Python Data Structures
Python offers several built-in data structures for organizing and storing data efficiently.

- **List** – Ordered and mutable collection.
- **Tuple** – Ordered and immutable collection.
- **Set** – Unordered collection of unique elements.
- **Dictionary** – Stores data in key-value pairs.

#### Introduction to Pandas
- Pandas is an open-source Python library used for data manipulation and analysis.
- It provides powerful data structures such as **Series** and **DataFrame**.
- Pandas makes it easy to clean, organize, filter, and analyze datasets efficiently.
