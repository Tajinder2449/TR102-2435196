### Learning Summary

#### Exploratory Data Analysis (EDA)
Exploratory Data Analysis (EDA) is the process of examining, summarizing, and visualizing datasets to understand their characteristics before applying statistical analysis or machine learning models. It helps identify patterns, relationships, anomalies, and data quality issues.

#### Basic EDA Techniques

**1. Understanding the Dataset**
- View the dataset structure.
- Check the number of rows and columns.
- Identify data types of each feature.

**2. Summary Statistics**
- Calculate measures such as mean, median, mode, minimum, maximum, and standard deviation.
- Understand the distribution and central tendency of data.

**3. Checking Missing Values**
- Identify null or missing values.
- Determine whether missing values need to be removed or imputed.

**4. Identifying Duplicate Records**
- Detect duplicate entries in the dataset.
- Remove duplicates to improve data quality.

**5. Analyzing Data Distribution**
- Study how values are distributed across features.
- Detect skewness and unusual patterns.

**6. Correlation Analysis**
- Measure relationships between numerical variables.
- Identify positively and negatively correlated features.

**7. Data Visualization**
Common visualizations used in EDA include:
- Histogram
- Box Plot
- Bar Chart
- Line Chart
- Scatter Plot
- Heatmap

These visualizations help in understanding trends, distributions, and relationships within the data.
